# lxmusic-xiaomi

[English](./README.md)

## 项目介绍

lxmusic-xiaomi 是一个基于 [lx-music-desktop](https://github.com/lyswhut/lx-music-desktop) 项目二次开发的项目，支持对接 [XiaoMusic](https://github.com/hanxi/xiaomusic) 项目，实现将播放设备改为你的小米设备。

## 功能特性

- 基于 lx-music-desktop v2.12.0 构建
- 支持对接 XiaoMusic 项目
- 将音频播放重定向到小米智能设备
- 保留 lx-music-desktop 的所有原有功能

## 版本信息

当前版本：**v1.0.0**

基于版本：lx-music-desktop v2.12.0
