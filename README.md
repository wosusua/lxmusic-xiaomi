# lxmusic-xiaomi

[中文说明](./README_zh.md)

## Introduction

lxmusic-xiaomi is a secondary development project based on [lx-music-desktop](https://github.com/lyswhut/lx-music-desktop). It integrates with the [XiaoMusic](https://github.com/hanxi/xiaomusic) project to enable playback on your Xiaomi devices.

## Features

- Based on lx-music-desktop v2.12.0
- Supports integration with XiaoMusic project
- Redirects audio playback to Xiaomi smart devices
- Maintains all original lx-music-desktop features

## Version
